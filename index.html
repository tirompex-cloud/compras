⚙️ Configurações

☁️ GitHub Gist (backup automático dos dados)

Cole seu GitHub Personal Access Token para ativar o backup automático. Cada lançamento e exclusão será sincronizado com um Gist privado na sua conta GitHub — seus dados ficam seguros mesmo se trocar de navegador.

1

Acesse github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)

2

Clique em Generate new token, marque apenas o escopo gist e clique em Generate

3

Copie o token e cole abaixo. Na primeira vez, um Gist privado é criado automaticamente.

O Gist é privado e acessível apenas pela sua conta. Nenhum dado fica exposto no repositório público.

📄 Google Sheets (planilha na nuvem)

Cole aqui a URL do seu Google Apps Script implantado. Cada lançamento será enviado automaticamente para a planilha Google Sheets.

1

Acesse script.google.com e crie um novo projeto

2

Cole o conteúdo do arquivo Code.gs (incluído no pacote baixado)

3

Clique em Implantar → Nova implantação, tipo App da Web, acesso "Qualquer pessoa"

4

Copie a URL gerada e cole abaixo

Abra sua planilha no Google Drive e copie a URL da barra de endereço do navegador.
A planilha é criada automaticamente na primeira vez que você salvar um lançamento. Ela ficará no seu Google Drive com o nome "📋 Lançamento de Compras".

🤖 API Anthropic (leitura de comprovantes)

Para que a IA leia imagens de comprovantes automaticamente, informe sua chave da API Anthropic. Obtenha em console.anthropic.com → API Keys.

Custo médio por comprovante: ~R$ 0,05. A chave fica salva apenas no seu navegador.
